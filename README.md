# SwiftUI-lookbackheisei

## 平成振り返りアプリ
平成最後のハッカソンで開発した[平成振り返りデバイス](https://vimeo.com/333354788)のiOSアプリ版。

## DEMO
アプリを起動して物理的に振り返ると平成のニュースやツイートが届きます。

![](image/lookbackheisei.gif)

## 開発環境
* macOS 10.15 Beta
* iOS13
* Xcode11 Beta
* Swift5

## アーキテクチャ
* MVVM

## 使用技術
* SwiftUI
* Alamofire
